GETWP^XPAR(): Return Word-Processing Text

Reference Type: Supported, Category: Toolkit-Parameter Tools Integration Agreement: 2263

Description

The PUT^XPAR API adds or updates a parameter instance and bypass the input transforms.

    REF: For descriptive information about the elements and how they are used in the callable entry points into XPAR, see the "Definitions" section.

Format

PUT^XPAR(entity,parameter[,instance],value[,.error])

Input/Output Parameters

See EN^XPAR

For the definition of the input and output parameters used in this API, see the EN^XPAR(): Add, Change, Delete Parameters API.

Example:

PUT^XPAR API-Example

>D PUT^XPAR("SYS","XPAR TEST MULTI FREE TEXT",0,"Good times",.ERROR)
