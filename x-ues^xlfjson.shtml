$$UES^XLFJSON(): Unescape JSON to a String

Reference Type: Supported, Category: Data Security, Integration Agreement: 6682

Description

This extrinsic function returns a unescaped string from a JSON format.

NOTE: This API was released with Kernel patch XU*8.0*680.

Format

$$UES^XLFJSON(x)

Input Parameters

x (required) A JSON escaped string to be unescaped.

Output

returns: Returns a unescaped string representation of the escaped JSON input string.

 

Example

>W $$UES^XLFJSON("\\one\\two\\three\\")
\one\two\three\
